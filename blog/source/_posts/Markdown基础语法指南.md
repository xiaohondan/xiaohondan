# Markdown基础语法指南

Markdown是一种轻量级标记语言，创建于2004年，现在已经成为世界上最流行的标记语言之一。它允许人们使用易读易写的纯文本格式编写文档，然后转换成有效的HTML文档。

## 为什么选择Markdown？

- **易于学习**：语法简单直观，没有复杂的格式要求
- **专注内容**：让你专注于写作内容，而不是排版格式
- **兼容性强**：几乎可以导出为任何格式（PDF, Word, HTML等）
- **跨平台**：在任何文本编辑器中都可以编写

## 基本语法

### 标题

使用`#`符号可以创建标题，`#`的数量表示标题的级别：

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

### 强调

```markdown
*斜体文本* 或 _斜体文本_
**粗体文本** 或 __粗体文本__
***粗斜体文本*** 或 ___粗斜体文本___
```

### 列表

无序列表可以使用`*`、`+`或`-`作为列表标记：

```markdown
* 第一项
* 第二项
* 第三项

+ 第一项
+ 第二项
+ 第三项

- 第一项
- 第二项
- 第三项
```

有序列表使用数字并加上`.`号：

```markdown
1. 第一项
2. 第二项
3. 第三项
```

### 链接

```markdown
[链接名称](链接地址)
[Google](https://www.google.com)
```

### 图片

```markdown
![替代文本](图片链接)
![Markdown Logo](https://markdown-here.com/img/icon256.png)
```

### 引用

在段落前使用`>`符号：

```markdown
> 这是一个引用。
> 
> 这是引用的第二段。
```

### 代码

行内代码使用反引号包裹：

```markdown
`print("Hello, World!")`
```

代码块使用三个反引号包裹，并可以指定语言：

````markdown
```python
def hello():
    print("Hello, World!")
hello()
```
````

### 表格

```markdown
| 表头1 | 表头2 | 表头3 |
| ----- | ----- | ----- |
| 单元格1 | 单元格2 | 单元格3 |
| 单元格4 | 单元格5 | 单元格6 |
```

### 分隔线

可以在一行中使用三个或更多的星号、减号或下划线来创建分隔线：

```markdown
***
---
___
```

## 进阶技巧

- 使用HTML标签：Markdown支持HTML标签
- 转义字符：使用反斜杠`\`转义特殊字符
- 任务列表：`- [ ]`表示未完成，`- [x]`表示已完成

## 结语

Markdown是一种非常实用的写作工具，掌握它可以让你的写作更加高效。希望这篇指南对你有所帮助！
